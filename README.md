# Analyse d’émotions dans des dialogues de jeux vidéo

## Présentation
Ce projet vise à entraîner un modèle de réseau de neurones capable de prédire l’émotion associée à une ligne de dialogue issue de jeux vidéo.

## Objectifs
- Collecter et préparer un corpus de dialogues de jeux vidéo
- Générer ou attribuer des étiquettes d’émotion aux dialogues
- Entraîner un modèle de classification d’émotions
- Évaluer les performances du modèle

## Corpus
Nous utilisons un corpus de dialogues de jeux vidéo :
https://github.com/seannyD/VideoGameDialogueCorpusPublic

Comme le corpus ne contient pas d’annotations émotionnelles, des méthodes automatiques de labellisation pourront être explorées.
